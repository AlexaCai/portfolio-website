myFlix web app (frontend)

A single page responsive web application for movies all fans
Overview

myFlix was created to serve as a reference in the domain of visual entertainment. Users can create an account and then log into myFlix to have access to information about different movies. They can search for movies, filter results based on different criteria and create lists of favorites.

Purpose

Learn and consolidate my skills with frontend React during my full-stack web development training to develop and showcase a user-friendly, appealing and complete web application (client and server side) built using JavaScript technologies and the MERN tech stack.

Objective

Develop an easy-to-use and responsive web app for users to access whenever they want to read and save information about different movies, as well as allowing me to put into practice my skills regarding the interaction between backend and frontend for a fully functional web app.
