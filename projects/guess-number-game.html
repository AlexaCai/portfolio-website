Guess the number web app

A simple retro game for people who want to pass the time by exercising their brain

Project brief

Overview

This web app was created to allow users to stimulate their brains by playing a simple yet pleasant game, in which they need to find a secret number. They can then share their highest score with friends if wanted.

Purpose

Consolidate my skills and my knowledge on JavaScript fundamentals / core concepts, and more specifically on DOM manipulation, conditional logics, event listeners, and functions creation.

Objective

Build a small game available online with an easy-to-use interface, simple UI elements, a pleasant overall visual and concise instructions that any user can follow.

Tech summary

This game is built with HTML, JavaScript, and CSS. Some tools, such as Google DevTools and Live-Server extension for example, have also been used during development.

  • JavaScript
  • CSS
