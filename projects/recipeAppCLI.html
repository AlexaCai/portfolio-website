Recipe app (CLI version)

A Python app to create, search and modify recipes

Project brief

Overview

This Recipe app (Command Line Interface version) was created to allow people to manage recipes. Users can create, read, modify and delete recipes, as well as search for recipes based on ingredients. For each recipe, users can add the recipe name, ingredients and cooking time, which then automatically calculate (based on the two latest elements) a difficulty level.

Purpose

Learn and consolidate my skills with Python fundamentals, data structures, object-oriented programming, and database interaction with Python, while creating a Recipe app CLI version that handles any common errors that may arise during users input or database access (e.g.: display user-friendly error messages and ensure the app can still run).

Objective

Build a Recipe app CLI version using Python and a MySQL database to provide an easy-to-use interface, supported by simple forms of input and concise instructions that any user can follow. Coding the app to ensure well-formatted codes according to standardized guidelines, supported by concise and helpful comments and documentation.

Tech summary

This Recipe app is built with Python version 3.8.7 and linked to a MySQL database using the object-relational mapper (ORM) SQLAlchemy. Even if not part of the final product, a connector package used to link the MySQL database to Python (allowing to perform operations on the database using SQL queries) has been tried out during development for testing purposes, as well as some logic to store and retrieve data by way of text files, binary files and pickles.
