Chat mobile app

A React Native mobile Chat app allowing users to talk and share medias

Project brief

Overview

Chat app was created to allow people to talk and exchange on a mobile app. Users can send messages like other very common chat apps (WhatsApp, Messenger, etc.). They can also send images, location and audios. Users can also customize their application by selecting the color background they want to use for the chat UI.

Purpose

Learn and consolidate my skills with React Native and React Native libraries to develop a user-friendly, appealing and easy-to-use chat mobile app, as well as working with Google Firestore (a NoSQL cloud-based database service offered by Google as part of its Firebase platform) to handle the chat app data.

Objective

Build a chat app for mobile devices using React Native and Google Firebase features to implement a database, storage space and anonymous authentication process. Coding the app to ensure users can send images, location and audios along with messages, access their messages when offline and visually customize their UI.

Tech summary

Chat app is built with React Native. Expo Go and Android Studio have been used for the development. The chat interfaces have been created using the Gifted Chat library. All chat messages are stored in a Google Firestore database and locally on usersâ€™ devices. Images and audios sent on Chat app are also stored in a Google Cloud storage created on Firebase, and the app authenticates users anonymously via Firebase Authentication. It also allows users to consult their messages offline thanks to client-side storage (AsyncStorage pack. from React Native).

My role

3 Implemented navigation logic between screens / UIs.

Retrospective


The original objective of this project was to create a mobile app allowing people to exchange messages, send images and audios and share location like other very common chat apps (WhatsApp, Messenger, etc.), which align with the final result.

Being in my first steps with React Native and mobile app development, I had to do some in-depth research and learn more about Expo Go, Android Studio and Firebase. However, these tools/platforms have very often rich and complete official documentation, which was very helpful.

I also already had a good base working with React (which presents similarities with React Native), which helped me, despite the new tools, technologies and topics emerging from this project, to navigate into the development process with a good and solid understanding of some of its key features.

One of the things I liked the most in this project was to learn more about the very useful features of Firebase, and being able to create a modern mobile app behaving very similarly to other very popular chat apps. I really enjoyed learning more about React Native, because it is a technology that, when mastered well, opens up a lot of possibilities for mobile projects.


Coding this project with a clear workflow also helped me to acquire a good understanding of the many new things encountered along the way regarding React Native, both in theory and in practice.

In the end, my hope with this Chat app is to give a simple, pleasant and easy-to-use experience to any users who wish to connect, organize activities or share beautiful moments.
