Professional website creation

A clear and modern website to present professional services

Project brief

Overview

The current website has been created to serve as an information resource for any companies, clients or professionals looking for a full-stack web development professional / or services. Users can know more about my past work experiences, as well as my current skills as a web developer.

Purpose

Consolidate my skills on designing a modern and appealing website in a professional context, using design technics such as wireframes conception, as well as working on the responsiveness aspect to make it fit to use on any devices - small or large.

Objective

Build a website with HTML, CSS, JavaScript and Bootstrap that showcase my professional experiences and skills. More particularly, ensuring users can easily consult any coding projects I've done so far, and reach out to me through different ways.

Tech summary

The website is built mostly with HTML and Bootstrap. Bootstrap was useful to ensure a consistent visual design, and ensure great responsiveness. CSS have also been used to modify some specific visual elements. As for JavaScript, it was mainly used to add smooth behaviors in the website, such as the scrolling effect between sections.

Tools used
  • HTML
  • CSS
  • JavaScript
  • Bootstrap

My role

Summary of my work
1 Created the wireframes for the website main page and specific project pages (in three responsive formats : for large / medium / small screens) to guide code implementation afterward
2 Created, structured and implemented codes in each file based on the wireframes
3 Used a combination of CSS and Bootstrap to create and ensure functional, responsive, theme-related and appealing visuals for the best user experience possible
4 Managed all interactions on Github (commits, pull requests, merging branches, etc.)
5 Performed several tests / debugging to ensure the proper functioning of the website
6 Deployed the web app online to make it accessible publicly
7 Built the complete website project case study to highlight the project's key information, development process and final results to showcase the work in a more accessible way for both developers and non-developers (current page)

Retrospective


The original objective of this project was to build a website that showcase my professional experiences and skills - and more particularly, that allow users to easily consult any coding projects I've done so far, which align with the final result.

I particularly liked the creative aspect of this work, and the design of the different wireframes. I quickly realized that to create an appealing and modern website, you need more than coding skills. It is also very important to think well in advance about what you want to present, why and how, in addition to anticipating what users will expect while consulting the website to design it accordingly and facilitate their experience.

Coding this project gave me the opportunity to acquire a good understanding on how to plan, conceptualize and then implement (coding) a website from start to finish, both in theory and in practice. With the knowledge I acquired along the way, I would feel equipped to work on a wide variety of other types of professional websites.


In the end, my hope with this website is that any user consulting it has a simple, pleasant and easy-to-use experience to get to know me professionally, and provide the means to reach out for any professional or work opportunities / inquiries.
