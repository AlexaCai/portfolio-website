To-Do-List web app

A To-Do-List web app that allows users to note and manage reminders
Overview

The To-Do-List web app has been created to allow in a very simple way users to (1) add items to a list, (2) mark items in the list as done, (3) delete items from the list and (4) change the order of items in the list.

Purpose

Consolidate my skills on working with jQuery library, considering I might eventually create bigger projects using jQuery or maintain / update existing projects / codes that use this library.

Objective

Build a To-Do-List web app that contains four different features using jQuery library. Learn how to convert vanilla JavaScript code to jQuery and understand their differences.

Tech summary

The To-Do-List web app relies entirely on jQuery and some HTML and CSS.

Tools used
  • jQuery
  • HTML
  • CSS

Retrospective


The original objective of this project was to build a To-Do-List web app that contains four different features using the jQuery library, which aligns with the final result.

Being in my first steps on working with jQuery, the most challenging thing was to understand the differences between JavaScript and this library. More precisely, to understand the functionalities jQuery offers on DOM-traversal, event-handling and Ajax.

I particularly appreciate working on this project since it allowed me to learn more about a very popular library for JavaScript. The knowledge acquired through this project could definitely come in handy in the future, whether for working on new projects, or on older projects using jQuery.

Building this project by converting an already existing vanilla JavaScript code for this web app into jQuery helped me to better understand the library both in theory and in practice, as well as his differences with JavaScript. However, even though I got a grasp of jQuery, I know that there is still much more to learn, which is why I remain very open on improving my skills with this library.


In the end, my hope with this web app was to create a simple, pleasant and easy-to-use product for anyone who wishes to not forget important things.
